# forum
I developed the backend of a forum application using Java with Spring Boot and Hibernate. 
The project uses a MySQL database and implements a full JWT-based authentication system.

The application includes complete CRUD operations for posts, allowing authenticated users to create, update, and delete posts.
Users must be authenticated in order to create a post. 
Additionally, I implemented comment functionality, enabling users to add comments to posts and view comments associated with each post.

Users can also view their own posts and track the comments they have made across different posts.
